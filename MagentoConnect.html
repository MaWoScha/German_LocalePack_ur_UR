Pakistanisch-Indisches Sprachpaket für Magento Commerce

(Magento Community Modules - Urdu (General) Language Pack)

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_ur_UR benutzen möchte, muss vorher den Ordner app/locale/fa_IR löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_ur_UR, primero debe eliminar la carpeta app/locale/fa_IR para que el paquete de idioma se puede instalar.
